## 🧭 Overview

**Nexora** is a smart OJT (On-the-Job Training) management system designed to simplify and secure internship tracking for students, teachers, and institutions.

It ensures that every log is:

* ✅ Accurate
* 📍 Location-verified
* 📸 Evidence-supported

---

## 🚀 Features

### 👨‍🎓 Student

* ⏱ Time In / Time Out system
* 📍 GPS-based location tracking
* 📸 Photo verification before submission
* 📊 Progress tracking (required vs completed hours)
* 📝 Daily activity logging

---

### 👩‍🏫 Admin / Coordinator

* 👥 Student management dashboard
* ✅ Approve / Reject submissions
* 📊 Reports and analytics
* 📢 Announcement system
* 📁 Document handling

---

### 🔐 System

* 🔑 Authentication (Supabase Auth)
* 🧠 Role-based access control
* ⚡ Real-time updates
* ☁️ Cloud storage integration

---

## 🎯 Core Concept

> Nexora ensures **trust and accountability** in OJT tracking.

By combining:

* 📍 Location verification
* 📸 Photo proof
* ⏱ Time tracking

We eliminate:

* ❌ Fake attendance
* ❌ Manual errors
* ❌ Unverified logs

## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/nexora.git
cd nexora
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```
